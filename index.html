1. Open your workspsace in C9.io
  2. Once loaded, go to Window > New Terminal
  3. Type git add {{file_name}}, where {{file_name}} is the name of the file you want to update.
  4. Once you've added your files, type git commit -m '{{message}}', where {{message}} is the message for your commit (e.g. Added new photos to index.html)
  5. Type git push to push code to GitHub.
